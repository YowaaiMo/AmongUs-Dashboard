# AmongUs Dashboard

Welcome to the AmongUs Dashboard! This interactive web application is designed to enhance your Among Us gaming experience by providing real-time information and statistics. The dashboard is built using HTML, CSS, and JavaScript to offer a seamless and user-friendly interface.

## Features

### 1. Real-time Player Status

Keep track of the status of players in your Among Us game. The dashboard displays whether players are "Alive," "Dead," or "Impostor." Updates are reflected instantly as the game progresses.

### 2. Task Progress

Stay informed about the completion status of tasks. The dashboard provides a visual representation of tasks that have been completed, helping players strategize and identify potential impostors.

### 3. Emergency Meetings

Track the number of emergency meetings used by players. This feature helps to manage the game dynamics and assess the level of communication within the crew.

### 4. Voting Results

View the results of voting rounds, including who was voted out and the number of votes each player received. This information is crucial for analyzing player behavior and making informed decisions.

### 5. Customizable Settings

Easily customize the settings to match your game preferences. Adjust the number of players, impostors, and other game parameters to ensure an accurate representation on the dashboard.

## How to Use

1. **Clone the Repository:**
   git clone https://github.com/your-username/amongus-dashboard.git

2. **Open `index.html` in your Browser:**
   Open the `index.html` file in your preferred web browser to launch the AmongUs Dashboard.

3. **Customize Settings (Optional):**
   Modify the `settings.js` file to customize game parameters such as the number of players, impostors, etc.

4. **Enjoy the Game:**
   Use the dashboard to stay on top of the game dynamics, collaborate with other players, and enhance your Among Us experience!
